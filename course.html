Laravel

Mastering Laravel for Web Development

Requirement

Basic knowledge of PHP, HTML, CSS, and JavaScript is required to follow this course. Familiarity with MVC architecture and object-oriented programming concepts will be beneficial but not mandatory.

Description

In this comprehensive Laravel course, you will dive deep into one of the most popular PHP frameworks for building robust web applications. From routing and controllers to advanced features like authentication, authorization, and API development, this course covers everything you need to become proficient in Laravel. Whether you’re building simple web applications or complex enterprise solutions, this course will equip you with the tools and best practices to develop scalable, maintainable, and efficient applications.

Module List

Modul 1: Introduction to Laravel Framework

Pengenalan dasar tentang Laravel, instalasi framework, dan arsitektur MVC.

Modul 2: Blade Templating Engine

Menggunakan Blade untuk mengelola tampilan di Laravel, termasuk directive dan inheritance.

Modul 3: Routing and Controllers

Memahami konsep routing di Laravel dan pembuatan controller untuk mengelola logika aplikasi.

Modul 4: Eloquent ORM and Database Management

Memahami bagaimana Eloquent ORM bekerja, relasi antar model, dan migrasi database.

Modul 5: Authentication and Authorization

Mengimplementasikan sistem login, registrasi, dan otorisasi pengguna di Laravel.

Modul 6: Form Handling and Validation

Membuat dan memvalidasi form di Laravel, menangani input pengguna, serta validasi server-side.

Modul 7: API Development with Laravel

Membuat RESTful API dengan Laravel, termasuk pengelolaan routing dan respon JSON.

Modul 8: Middleware and Security

Menerapkan middleware untuk keamanan aplikasi dan cara melindungi aplikasi dari serangan umum seperti CSRF dan XSS.

Modul 9: Deploying Laravel Applications

Mempelajari cara mendepoy aplikasi Laravel ke server produksi menggunakan tools seperti Forge atau Heroku.

Modul 10: Task Scheduling and Queues

Belajar tentang task scheduling untuk menjalankan tugas secara otomatis dan pengelolaan antrian untuk meningkatkan performa aplikasi.

Modul 11: Testing in Laravel

Memahami cara menulis dan menjalankan unit test, feature test, serta menggunakan Laravel Dusk untuk testing aplikasi berbasis browser.

Modul 12: Advanced Laravel Features

Menguasai fitur-fitur lanjutan Laravel seperti Event Broadcasting, Jobs, dan real-time notifications dengan Laravel Echo.
